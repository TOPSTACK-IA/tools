Guia de Deploy Git

Instruções de atualização via PowerShell

1 Acessar a Pasta

Navegue até o diretório onde o projeto está salvo no seu computador.

cd "C:\[CAMINHO_DA_SUA_PASTA]\[NOME_DO_PROJETO]"
2 Adicionar Arquivos (Stage)

Este comando prepara todas as suas alterações para serem salvas.

git add .
3 Salvar Versão (Commit)

Cria um pacote fechado com as mudanças e uma mensagem de identificação.

git commit -m "Atualizacao do site"
4 Enviar (Push Padrão)

Envia as atualizações para o GitHub. Use este comando no dia a dia.

git push
5 Envio Forçado (Force Push) Zona de Perigo

Atenção: Use apenas se o GitHub rejeitar o envio padrão por conflito de histórico. Este comando apaga o que está na nuvem e substitui pelo seu PC.

git push -u origin main --force
